Reports usage of deprecated global settings 'Default Tags' and 'Force Tags' in Robot Framework files.

These global settings have been deprecated since Robot Framework 6.1.0:

  • Force Tags: Should be replaced with the new Test Tags setting
  • Default Tags: Has been removed without replacement

Example of deprecated usage: 

*** Settings ***
Force Tags      smoke
Default Tags    regression

Should be updated to: 

*** Settings ***
Test Tags       smoke
# Default Tags has no replacement - remove or handle differently

This inspection helps maintain compatibility with newer versions of Robot Framework by identifying outdated global setting names.
