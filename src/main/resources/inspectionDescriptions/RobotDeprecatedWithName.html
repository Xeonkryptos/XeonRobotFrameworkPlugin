Reports usage of the deprecated 'WITH NAME' suffix in Robot Framework library imports.

The 'WITH NAME' suffix was deprecated in Robot Framework 6.0.0 and should be replaced with the 'AS' suffix for better consistency and readability.

Example of deprecated syntax: 

Library    MyLibrary    WITH NAME    MyLib

Should be replaced with: 

Library    MyLibrary    AS    MyLib

This inspection only triggers when Robot Framework version 6.0.0 or higher is detected in your project.
