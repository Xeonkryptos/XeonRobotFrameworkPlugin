Reports unused keyword-only markers in Robot Framework keyword arguments.

In Robot Framework, a keyword-only marker (represented by an empty variable ${}) is used to separate positional arguments from keyword-only arguments. However, if no arguments with default values follow the keyword-only marker, it serves no purpose and should be removed.

Example:

*** Keywords ***
My Keyword
    [Arguments]    ${arg1}    ${}    # <-- unnecessary marker
    Log    ${arg1}

My Other Keyword
    [Arguments]    ${arg1}    ${}    ${arg2}=default    # <-- marker is needed
    Log    ${arg1}    ${arg2}

This inspection helps maintain clean and readable Robot Framework code by identifying and highlighting redundant keyword-only markers.
