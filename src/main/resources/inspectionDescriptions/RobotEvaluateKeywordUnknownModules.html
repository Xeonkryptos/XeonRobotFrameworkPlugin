Reports unknown Python modules passed to the Robot Framework Evaluate keyword.

This inspection detects unresolved module names in the comma-separated modules argument, including the second positional argument form of Evaluate. It helps catch misspelled module names, missing dependencies, and PYTHONPATH issues before the test is executed.

Example: 


*** Test Cases ***
Example
    ${value}=    Evaluate    json.loads('{"enabled": true}')    modules=json,jsno

The inspection highlights each unknown module name inside the argument value. Use valid, importable Python module names and make sure they are available in the current project environment.
