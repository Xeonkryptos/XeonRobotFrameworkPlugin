Reports imports of JSON variable files when the configured Robot Framework version does not support this format.

JSON variable files were introduced in Robot Framework 6.1. This inspection highlights import statements that reference .json variable files when the project’s Robot Framework version is set to an earlier version, helping you avoid runtime errors due to unsupported file formats.

Example of a problematic import: 


*** Settings ***
Variable    variables.json

To resolve this issue, either upgrade your Robot Framework version to 6.1 or higher, or convert the JSON variable file to a supported format such as .robot or .py.
