Detects deprecated keyword settings in Robot Framework code and suggests modern alternatives.

This inspection identifies the deprecated `[Return]` setting in user keywords, which has been superseded by the `RETURN` statement since Robot Framework 7.0.0. Using the modern syntax improves code readability and follows current best practices.

Example of deprecated usage: 

My Keyword
    [Arguments]    ${arg}
    [Return]       ${result}    # deprecated

Should be replaced with: 

My Keyword
    [Arguments]    ${arg}
    RETURN         ${result}    # modern syntax

This inspection only triggers when Robot Framework version 7.0.0 or later is detected in the project.
